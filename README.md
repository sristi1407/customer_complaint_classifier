# Customer Complaint Classifier

Multi-class text classification using CFPB Consumer Complaint Database.

## Dataset
- 366,945 real customer complaints
- 18 financial product categories
- Downloaded from: https://www.kaggle.com/datasets/selener/consumer-complaint-database

## Setup

1. Download `rows.csv` from Kaggle
2. Place in `data/raw_data/rows.csv`
3. Run preprocessing notebook
4. Train models on Sristi1 branch
